# UK Population Interactive Pyramid

This project provides a high-performance, interactive visualization of the United Kingdom's population demographics. It transforms raw census data into an intuitive population pyramid, allowing users to track demographic shifts over time.

## Purpose
The goal of this application is to provide a clear, visual representation of how the UK's age distribution has changed. By comparing male and female population segments side-by-side, researchers and enthusiasts can identify trends such as aging populations or shifts in birth rates.

## Key Features
*   **Custom Data Integration**: Use the `merge_data.py` utility to process raw Excel census files and create your own unified DataFrame automatically.
*   **Dynamic Interactive Diagram**: Explore the data using a real-time slider that updates the pyramid instantly without flickering, thanks to optimized artist-reuse techniques.
*   **Animation Export**: High-quality animation support allows you to save the entire chronological sequence as a GIF for use in presentations or reports.

## Getting Started
1.  **Prepare Data**: Place your census `.xlsx` files into the `src/census` folder.
2.  **Run the Application**:
    ```bash
    python src/main.py
    ```
3.  **Interact**:
    *   Type `n` when prompted to open the **Interactive Mode** with the year slider.
    *   Type `y` to enter **Export Mode** and save your animation to the `results/` folder.

## Requirements
*   Python 3.x
*   Matplotlib
*   Pandas
*   Openpyxl (for Excel processing)
